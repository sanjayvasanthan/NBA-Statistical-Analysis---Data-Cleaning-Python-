# NBA-Statistical-Analysis---Data-Cleaning-Python-

## Project Overview
Analysis of NBA scoring and 3-point shooting trends from 2003-2023.

## 📁 Files in this Repository
- `NBA_Data_Cleaning.ipynb` - Jupyter notebook with complete data cleaning and analysis
- `NBA_Project_Cleaned_Data_2003-2023.xlsx` - Final cleaned dataset (2003-2023)
- `NBA_Project_Cleaned_Data_2003-2025.xlsx` - Extended dataset including recent seasons
- `uncleaned_nba_dataset.xlsx` - Original raw data before cleaning

## 🛠️ Tools Used
- Python (pandas)
- Jupyter Notebook
- Excel
